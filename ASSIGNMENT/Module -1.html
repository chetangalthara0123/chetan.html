9.How is Cell Padding different from Cell Spacing? With Example.

ANS:- 1. Cell Padding: Cell padding is the space between the content of a table cell and its border. It is controlled by the cellpadding attribute in the table tag. You can specify the amount of padding in pixels or as a percentage of the cell width. 2. Cell Spacing: Cell spacing is the space between adjacent cells in a table. It is controlled by the cellspacing attribute in the table tag. Like cell padding, you can specify the spacing in pixels or as a percentage of the cell width.

16. How are active links different from normal links?

ANS:-1. Normal Links: Normal links are links that are in their default state, meaning they haven't been interacted with by the user. They typically have a default appearance specified by the browser or any styles applied in the CSS. Normal links are displayed when the page is initially loaded 2. Active Links: Active links refer to links that are currently being interacted with by the user. This state is often triggered when the user clicks on the link. The appearance of active links can be styled differently to provide feedback to the user that the link is being activated. The styling for active links is often specified using the :active pseudo-class in CSS.

18. What is SVG?

ANS:- SVG stands for Scalable Vector Graphics. It is a widely used XML-based file format for describing two-dimensional vector graphics. Key features of SVG include: 1.Scalability: SVG images can be scaled to any size without losing quality. This makes them suitable for a variety of applications, from small icons to large graphics. 2.XML Format: SVG files are written in XML (eXtensible Markup Language), a text-based format that is both human-readable and machine-readable. This makes SVG files easy to create, modify, and manipulate. 3.Interactivity: SVG supports interactivity and animation. You can use JavaScript to add interactive elements to SVG graphics, making them dynamic and engaging. 4.Accessibility: SVG supports accessibility features, making it possible to create graphics that are friendly to users with disabilities. This is particularly important for web design and other applications where accessibility is a priority. 5. Text-Based: Being text-based, SVG files can be compressed, making them smaller in file size. This can lead to faster loading times for web pages and reduced bandwidth usage. 6.Wide Browser Support: Most modern web browsers support SVG, making it a popular choice for web design and development.
